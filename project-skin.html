Skin Disease Diagnosis App

AI-powered image analysis for faster diagnosis.

Description:

This Flask-based web application allows users to upload an image of a skin condition and receive an AI-generated prediction and treatment advice. It integrates with a third-party AI API to analyze dermatological images.

Technologies:
