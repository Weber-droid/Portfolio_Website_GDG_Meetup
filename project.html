Bug Tracker API

Manage bugs and track issues with ease.

Description:

This project is a RESTful API built using Express.js and MongoDB. It allows users to create, read, update, and delete bug reports. It supports authentication and role-based access for developers and managers.

Technologies:

  • Node.js
  • JavaScript
  • MongoDB
